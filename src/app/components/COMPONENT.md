# Components

Créer les composant suivants :

- ✅ Un composant `Header` qui affiche un titre
- ✅ Un composant `Footer` qui affiche des informations de copyright.
- ✅ Un composant `Main` qui affiche un article avec un titre et du contenu.
- ✅ Un composant `Button` qui affiche un bouton cliquable avec un texte.

- Un composant `Card` qui affiche  une carte avec :
> https://getbootstrap.com/docs/5.3/components/card/
- une image  : https://picsum.photos/200
    - bonus faire un effet de zoom au survol de l'image
- un titre
- une description 
- accepte un ou plusieurs boutons `app-button`.

- Un composant `Section` qui affiche une section avec un titre et zone d'éléments.
- Un composant `Navigation` qui affiche un menu de navigation.
